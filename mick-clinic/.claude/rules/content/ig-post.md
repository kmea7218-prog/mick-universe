# IG 貼文規則

## 受眾
- 主:牧場主、酪農工作者、獸醫同業
- 次:對畜牧 / 動物有興趣的一般讀者

## 格式
- **首圖**:1 張強照片 / 圖表(不要純文字卡)
- **標題**:8–15 字,避免釣魚標(「你不知道的...」禁用)
- **正文**:300 字內;一段一個重點
- **結尾**:一個問題或 CTA(留言、私訊、存起來)

## 內容類型
1. **個案故事**:一頭牛、一個牧場的真實故事(見 `storytelling.md`)
2. **知識科普**:一個繁殖 / 營養 / 疾病概念(見 `education.md`)
3. **數據觀察**:從資料看出的趨勢(匿名化)
4. **幕後**:診療日常、工具、方法

## Hashtag
固定前綴:`#mick診所 #乳牛獸醫 #牧場資料`
主題:2–3 個(`#繁殖管理` `#DHI` `#乳房炎` 等)
流量 tag 不超過 5 個。

## 禁止
- 不要把客戶牧場的可識別資訊(場名、人名、牛號)寫出來
- 不要在未確認下引用數據
- 不要用誇大標題(「所有牧場都該...」、「99% 獸醫不知道」)
