# KPI 定義總表

所有 dashboard、報表、對客戶的數字都以此檔為準。**不要在程式碼或報表裡重新定義公式**,一律引用這裡。

## 繁殖類
見 `reproduction-metrics.md`。

## 產乳類
| KPI | 定義 | 單位 |
|---|---|---|
| 日均乳量 | 近 30 天平均日乳量 | kg/頭/日 |
| 305 天乳量 | 泌乳期前 305 天累積乳量 | kg |
| 乳脂率 | 當月 DHI 乳脂 | % |
| 乳蛋白 | 當月 DHI 乳蛋白 | % |
| SCC 幾何平均 | 體細胞數幾何平均 | 千/mL |

## 健康類
| KPI | 定義 |
|---|---|
| 乳房炎發生率 | 當月新發乳房炎牛隻 / 泌乳牛群 |
| 蹄病發生率 | 當月新發蹄病牛隻 / 總牛隻 |
| 酮症比例 | DHI 脂蛋比 > 1.5 的比例 |

## 營運類
| KPI | 定義 |
|---|---|
| 牛群週轉率 | 年度新進牛 / 平均牛群 |
| 乾乳牛比例 | 乾乳牛 / 總成牛 |
| 每頭牛每日利潤 | (乳收入 − 飼料成本 − 其他直接成本) / 在群牛日 |

## 變更規則
改動公式要:
1. 在 PR / commit message 寫明原因
2. 更新 dashboard 上的定義提示
3. 重算歷史資料並標註「重算日期」
